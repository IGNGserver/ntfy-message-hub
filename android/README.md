# 讯笺 Android

讯笺的 Jetpack Compose Material 3 Android 客户端。

## Features

- First-launch server configuration
- Access-key login
- Channel browsing
- Timeline / grouped views
- Tag filter dialog with scrolling multi-select
- Settings for dynamic theme, vibration, server config, local storage, sync mode
- Offline banner with cached message fallback
- Manual sync support with sync status

## Build

```bash
cd android
./gradlew :app:assembleDebug
```
